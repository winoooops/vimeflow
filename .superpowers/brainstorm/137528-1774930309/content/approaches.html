Chat View — Component Architecture

How should the Chat center column be decomposed into React components?

A

Flat Components

Minimal component tree — a single ChatView page component containing MessageThread, UserMessage, AgentMessage, CodeBlock, MessageInput, and StatusBadge. Simple, matches the design 1:1.

B

Design System First

Build a primitive component library first (Surface, GlassPanel, Badge, Avatar, Typography), then compose the Chat view from those primitives.

C

Hybrid — Tokens + Flat Components (Recommended)

Set up the Tailwind theme config with all design tokens (colors, fonts, border-radius) from the reference HTML, then build flat Chat components using those tokens. No component abstractions, but the design system vocabulary is baked into Tailwind.
