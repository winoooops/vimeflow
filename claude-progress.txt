## Known Issues & Blockers

### Bash Hook Restriction

**Issue**: A PreToolUse:Bash hook in `~/.claude/settings.json` blocked all Bash commands during this session with error: "PreToolUse:Callback hook blocking error from command: 'callback': Empty command"

**Hook Configuration**:
```json
{
  "hooks": {
    "PreToolUse": [{
      "matcher": "Bash",
      "hooks": [{
        "type": "command",
        "command": "npx block-no-verify@1.1.2"
      }]
    }]
  }
}
```

**Impact**: Could not execute:
- `git init` - Git repository initialization
- `npm install` - Dependency installation
- `chmod +x` - Making scripts executable
- `npm run prepare` - Husky setup
- Any verification commands

**Workaround Applied**: Created all files with correct content and provided SETUP.md with manual instructions for running the blocked commands.
