# Design archive — historical handoffs & mockups

These directories and files were point-in-time **design handoffs**, migration
briefs, Google Stitch `code.html` mockups, and runnable React prototypes. They
predate (and drifted from) the shipped app — different layouts (4/5-zone + icon
rail), pre-#442 surface tokens, dark-only hex, and screens that no longer exist.

**They are no longer the source of truth.** The single source of truth is the
consolidated, code-grounded spec in [`../UNIFIED.md`](../UNIFIED.md) (with
[`../DESIGN.md`](../DESIGN.md) as the foundation and `src/theme/themes/*.ts` as
the runtime token SSoT). Every still-valid component contract from these files
has been folded into `UNIFIED.md`.

Kept here as a **backup / visual reference only**, pending deletion in a
follow-up once the consolidated spec is confirmed in production use.

> Do **not** derive new work from anything in this folder. Use `../UNIFIED.md`.
